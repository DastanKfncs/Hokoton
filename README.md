# Сайт голосований

Простой рабочий сайт для создания и проведения голосований.

## Запуск

1. Откройте терминал в папке проекта.
2. Установите зависимости:

```bash
npm install
```

3. Запустите сайт:

```bash
npm start
```

4. Откройте в браузере:

```
http://localhost:3000
```

## Возможности

- Просмотр списка голосований
- Голосование за вариант
- Мгновенное обновление результатов
- Сохранение голосов в файл `data/polls.json`
